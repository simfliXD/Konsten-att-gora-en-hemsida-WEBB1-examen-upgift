# Konsten att skapa en hemsida

En webbsida som visar grunderna i webbutveckling med HTML, CSS och JavaScript.  
Sidan innehåller bland annat avsnitt om webbhistoria, lagar på nätet, HTML, CSS och JavaScript.

## Innehåll
- `index.html` - startsida
- `historia.html` - webbens historia
- `lagar.html` - lagar och regler på nätet
- `html.html` - HTML
- `css.html` - CSS
- `javascript.html` - JavaScript
- `style.css` - gemensam styling
- `script.js` - gemensama JavaScript-funktioner

## Hur man kör projektet

### Alternativ 1: Öppna direkt
Öppna `index.html` i din webbläsare.
När man öppnar documentet direkt så fungerar ej localStorage mellan sidorna.

### Alternativ 2: Kör en lokal server
Om du vill köra projektet via en lokal server kan du använda till exempel Python:
Kör från projektmappen:
