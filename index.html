Vad lär du dig här?

Den här webbsidan är en guide till webbutveckling. Vi går igenom allt från grunderna.

Du hittar förklaringar av alla tre tekniker i menyn ovan. Den här sidan är i sig själv gjord som ett exempel – titta gärna på källkoden med F12 i webbläsaren!

Hur fungerar en webbsida egentligen?

En typisk webbsida byggs upp av tre olika tekniker:

  • HTML – strukturen. HTML bestämmer vad som finns på sidan: rubriker, text, bilder, knappar och länkar.
  • CSS – utseendet. CSS styr hur allt ser ut: färger, typsnitt, layout och animationer.
  • JavaScript – interaktiviteten. JavaScript gör sidan levande: den reagerar på knapptryck, hämtar data och uppdaterar innehållet utan att ladda om sidan.

När du skriver in en webbadress i din webbläsare så sker det faktiskt en del bakom kulisserna:

  1. Webbläsaren frågar en DNS-server om vilken IP-adress domännamnet (t.ex. google.com) pekar på.
  2. Webbläsaren och servern skapar en TCP anslutning över port 443 för HTTPS (säker kommunikation).
  3. Webbläsaren frågar servern om filerna till webbsidan, vilket sedan skickas till webbläsaren.
  4. Webbläsaren läser och tolkar HTML, CSS och JavaScript koden.
  5. Sedan renderas sidan och du kan se den på skärmen.

Det här sker ofta på bara millisekunder varje gång du laddar en sida.
