JavaScript

Vad är JavaScript?

JavaScript (förkortat JS) är ett programmeringsspråk som gör webbsidor interaktiva. Medan HTML strukturerar innehållet och CSS styr utseendet, är det JavaScript som ger sidan liv – till exempel genom att reagera på knapptryck, hämta data från servrar, animera element, eller som på den här sidan, byta mellan mörkt och ljust tema.

JavaScript körs direkt i webbläsaren utan att behöva installeras. Det är ett av de mest använda programmeringsspråken i världen och är grunden för moderna webbapplikationer.

JavaScript läggs till på en sida med <script>-taggen, helst längst ned i <body> eller med attributet defer i <head>: 

<!-- Extern JS-fil, defer = vänta tills HTML är inläst -->
<script src="script.js" defer></script>

<!-- Eller inline direkt i HTML -->
<script>
  console.log("Hej världen!");
</script>

Variabler och datatyper

Variabler är namngivna behållare för data. I modern JavaScript använder man let för värden som kan ändras och const för värden som inte ska ändras. Undvik det äldre var. 

const namn = "Anna";          // Sträng (text)
let ålder = 25;               // Tal
let loggadIn = true;          // Boolean (sant/falskt)
const färger = ["röd", "blå"]; // Array (lista)

// const kan inte tilldelas om:
// namn = "Björn"; // FEL!

// let kan tilldelas om:
ålder = 26; // OK

Villkor (if/else)

const timme = new Date().getHours();

if (timme < 12) {
  console.log("God morgon!");
} else if (timme < 18) {
  console.log("God eftermiddag!");
} else {
  console.log("God kväll!");
}

Loopar

// Gå igenom en lista
const frukter = ["äpple", "banan", "päron"];

for (const frukt of frukter) {
  console.log(frukt);
}
// Skriver ut: äpple, banan, päron

Funktioner

En funktion är ett namngivet kodblock som kan anropas när som helst. Det gör att du slipper skriva samma kod om och om igen. 

// Definiera en funktion
function hälsa(namn) {
  return "Hej, " + namn + "!";
}

// Anropa funktionen
const meddelande = hälsa("Maria");
console.log(meddelande); // "Hej, Maria!"

Modern JavaScript använder ofta så kallade pilfunktioner: 

// Pilfunktion – kortare syntax
const hälsa = (namn) => "Hej, " + namn + "!";

// Med flera rader
const summa = (a, b) => {
  const resultat = a + b;
  return resultat;
};

DOM – Manipulera sidan med JavaScript

DOM (Document Object Model) är webbläsarens representation av HTML-sidan som ett träd av objekt. Med JavaScript kan du hämta, ändra, lägga till och ta bort HTML-element och deras innehåll i realtid.

Hämta element

// Hämta ett element med ID
const knapp = document.getElementById("min-knapp");

// Hämta element med CSS-selektor (returnerar det första)
const rubrik = document.querySelector("h1");

// Hämta alla element som matchar (returnerar en lista)
const alla = document.querySelectorAll(".kort");

Ändra innehåll och stil

const rubrik = document.querySelector("h1");

// Ändra textinnehållet
rubrik.textContent = "Ny rubrik!";

// Ändra CSS-stil direkt
rubrik.style.color = "red";
rubrik.style.fontSize = "3rem";

// Lägg till eller ta bort CSS-klasser (rekommenderat)
rubrik.classList.add("aktiv");
rubrik.classList.remove("gammal");
rubrik.classList.toggle("mörk"); // Lägg till om den saknas, ta bort om den finns

Skapa och lägga till element

// Skapa ett nytt element
const nyParagraf = document.createElement("p");
nyParagraf.textContent = "Det här är ny text!";
nyParagraf.classList.add("info");

// Lägg till det i DOM:en
document.body.appendChild(nyParagraf);

Händelselyssnare

Händelselyssnare (event listeners) låter JavaScript reagera på saker som händer på sidan – som att användaren klickar, skriver eller scrollar. 

const knapp = document.getElementById("min-knapp");

// Kör en funktion när knappen klickas
knapp.addEventListener("click", () => {
  alert("Du klickade på knappen!");
});

// Lyssna på tangenttryckningar
document.addEventListener("keydown", (event) => {
  if (event.key === "Escape") {
    console.log("Escape trycktes!");
  }
});

// Lyssna när ett formulärfält ändras
const input = document.querySelector("input");
input.addEventListener("input", (event) => {
  console.log("Värdet är nu:", event.target.value);
});

Mörkt läge med JavaScript

Mörkt läge är ett bra praktiskt exempel på hur HTML, CSS och JavaScript samverkar. Idén är enkel: JavaScript lägger till eller tar bort en CSS-klass på <body>, och CSS definierar hur sidan ska se ut med den klassen.

Steg 1 – HTML: en toggle-knapp

<button id="tema-knapp">🌙 Byt tema</button>

Steg 2 – CSS: definiera mörkt läge

/* Ljust läge (standard) */
body {
  background-color: white;
  color: black;
}

/* Mörkt läge – aktiveras av klassen dark-mode på body */
body.dark-mode {
  background-color: #1a1a2e;
  color: #e0e0e0;
}

Steg 3 – JavaScript: byt tema vid klick + spara val

const knapp = document.getElementById("tema-knapp");

// Kolla om användaren har ett sparat val från förra besöket
const sparat = localStorage.getItem("tema");
if (sparat === "mörkt") {
  document.body.classList.add("dark-mode");
}

knapp.addEventListener("click", () => {
  // Växla klassen dark-mode
  document.body.classList.toggle("dark-mode");

  // Spara valet i localStorage så det finns kvar nästa gång
  const ärMörkt = document.body.classList.contains("dark-mode");
  localStorage.setItem("tema", ärMörkt ? "mörkt" : "ljust");
});

localStorage är en inbyggd webbläsarfunktion som låter dig spara data som finns kvar även när sidan stängs. På den här webbsidan används exakt samma teknik för att komma ihåg vilket tema du valde.

Komplett interaktivt exempel

Här är ett komplett litet exempel som kombinerar allt ovan – en enkel räknare: 

<!-- HTML -->
<div id="raknare">
  <p>Antal klick: <span id="antal">0</span></p>
  <button id="oka">+ Öka</button>
  <button id="nollstall">Nollställ</button>
</div>
/* CSS */
#raknare {
  text-align: center;
  padding: 2rem;
  font-size: 1.5rem;
}

#raknare button {
  padding: 0.5rem 1.5rem;
  margin: 0.5rem;
  font-size: 1rem;
  cursor: pointer;
  border-radius: 6px;
  border: none;
  background-color: #ff5a09;
  color: white;
}
// JavaScript
let antal = 0;
const antalSpan = document.getElementById("antal");

document.getElementById("oka").addEventListener("click", () => {
  antal++;
  antalSpan.textContent = antal;
});

document.getElementById("nollstall").addEventListener("click", () => {
  antal = 0;
  antalSpan.textContent = antal;
});
